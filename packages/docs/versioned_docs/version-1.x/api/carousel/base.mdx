# Carousel

Carousel is a component which can scroll(re render) item

```jsx
import { Carousel } from "@tlgr/carousel";
const carousel = new Carousel(options);
```

Events described at [events](./events) page

## extends

- [InlineComponent](../component/inline)

## `constructor(bot, options)`

- `bot` - <[Object][object]>. [Telegraf](https://telegraf.js.org/classes/Telegraf.html) instance.
- `options` - <[Object][object]>. Carousel options.

  - `type` - <[string][string]>. can be the one of the next items: `'photo' | 'audio' | 'animation' | 'video' | 'document'`
  - `items` - <[object][object] | [string][string]> - Carousel item. Usually it is a url to content. E.g. `https://example.com/path/to/image`
  - `mode` - <[string][string]>. Reply mode. `new` - sends new message. `edit` - edit first sended message. Default is `edit`
  - `loop` - <[boolean][boolean]>. Loop mode. Starts over again when end item is reach out. Default is `false`
  - `startFrom` - <[number][number]> - Index which carousel will start. Default is `0`.
  - `controls` - <[object][object]>. Controls decoration object. Override prev and next button behavior.
    - `prev` - <[object][object]>. Previous button decoration.
      - `symbol` - <[string][string]>. Previous symbol. Default is `<`.
      - `alert` - <[object][object]>. Alert options.
        - `text`- <[string][string]>. Alert text.
        - `mode` - <[string][string]>. Alert mode. Possible values: `text` or `modal`. Default is `text`
        - `show` - <[boolean][boolean]>. Show modal or not. Default is `false`.
    - `next` - <[object][object]>. Next button decoration.
      - `symbol` - <[string][string]>. Previous symbol. Default is `>`.
      - `alert` - <[object][object]>. Alert options.
        - `text`- <[string][string]>. Alert text.
        - `mode` - <[string][string]>. Alert mode. Possible values: `text` or `modal`. Default is `text`
        - `show` - <[boolean][boolean]>. Show modal or not. Default is `false`.

## methods

### send(context)

<small> added in 1.1.0</small>

Sends carousel from bot to user

- `context` - <[Object][object]>. Telegraf [context](https://telegraf.js.org/classes/Context.html) instance.

### render()

<small> added in 1.1.0</small>

**Note:** this method are come from inheritance. This method returns `prev` and `next` buttons. Use `send` method instead.

## properties

### item

Return current render item.

### index

return current render item index in array which starts from `0`.

### length

Carousel items length.
