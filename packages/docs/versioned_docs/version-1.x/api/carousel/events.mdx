# Events

### Sent

<small>added in 1.1.0</small>

Fires when carousel sent.

#### name

`sent`

#### payload

- `ctx` - <[Object][object]>. Telegraf update [Context](https://telegraf.js.org/classes/Context.html#addStickerToSet).

### Next

<small>added in 1.1.0</small>

#### name

`next`

#### payload

- `ctx` - <[Object][object]>. Telegraf update [Context](https://telegraf.js.org/classes/Context.html#addStickerToSet).
- `instance` - <[Object][object]>. Carousel instance.

### Previous

<small>added in 1.1.0</small>

#### name

`prev`

#### payload

- `ctx` - <[Object][object]>. Telegraf update [Context](https://telegraf.js.org/classes/Context.html#addStickerToSet).
- `instance` - <[Object][object]>. Carousel instance.
