# Poll

Poll button is a keyboard only button. After clicking - call internal telegram poll builder.

After submitting the poll - fires [`Answer anonymous`](#answer-anonymous) or [`Answer user`](#answer-user) events.

## extends

[KeyboardButton](./keyboard)

## Events

### Click

<small>added in 1.1.0</small>

#### name

`click`

#### payload

- `ctx` - <[Object][object]>. Telegraf update [Context](https://telegraf.js.org/classes/Context.html#addStickerToSet).
- `button` - <[Object][object]> - the [Keyboard Button](./keyboard) instance

### Answer anonymous

<small>added in 1.1.0</small>

#### name

`answer_anonymous`

#### payload

- `poll` - <[Object][object]> - the [poll](https://telegraf.js.org/classes/Context.html#poll) object
- `instance` - <[Object][object]>. [Button](#Poll) isntance.


### Answer user

<small>added in 1.1.0</small>

#### name

`answer_user`

#### payload

- `pollAnswer` - <[Object][object]> - the [pollAnswer](https://telegraf.js.org/classes/Context.html#pollAnswer) object
- `instance` - <[Object][object]>. [Button](#Poll) isntance.
