# Phone

Phone button is a keyboard only button. After clicking - ask app to share phone data with bot.

After location recieved - fires [`contact`](#contact) event.

## extends

[KeyboardButton](./keyboard)

## Events

### Click

<small>added in 1.1.0</small>

#### name

`click`

#### payload

- `ctx` - <[Object][object]>. Telegraf update [Context](https://telegraf.js.org/classes/Context.html#addStickerToSet).
- `button` - <[Object][object]> - the [Keyboard Button](./keyboard) instance

### Contact

<small>added in 1.1.0</small>

#### name

`click`

#### payload

- `ctx` - <[Object][object]>. Telegraf update [Context](https://telegraf.js.org/classes/Context.html#addStickerToSet).
- `contact` - <[Object][object]> - the [phone](https://telegraf.js.org/modules/Types.html#ExtraContact) object
