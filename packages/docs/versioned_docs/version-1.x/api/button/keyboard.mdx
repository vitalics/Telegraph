# Keyboard

Button which renders in keyboard location.

## extends

[KeyboardComponent](../component/keyboard)

## constructor(bot)

<small>added in 1.1.0</small>

- `bot` - <[Object][object]>. [Telegraf](https://telegraf.js.org/classes/Telegraf.html) instance.
- `name` - <[string][string]>. Button name, or text which user will see.
- `options?` - <[Object][object]>
  - `prefix` - <[string][string]>. Name of the control. Default is `@tlgr/button`
  - `disableUUID` - <[boolean][boolean]>. Enable/Disable uuid generation for button control. It guarantee about button has be unique name. Default is `false`
  - `alert` - <[Object][object]>. Alert options. Default is `undefined`
    - `text`- <[string][string]>. Alert text.
    - `mode` - <[string][string]>. Alert mode. Possible values: `text` or `modal`. Default is `text`
    - `show` - <[boolean][boolean]>. Show modal or not. Default is `false`.

## properties

### `uuid`

<small>added in 1.1.0</small>

#### modifiers

- readonly

#### description

Random generated hash. Returns `undefined` in case if `options.disableUUID=true`

### `name`

<small>added in 1.1.0</small>

#### modifiers

- readonly
- getter

#### description

Resolved button name. This property consist of `options.prefix` and `options.disableUUID`.
