# Analytics

## `constructor(bot, [...plugins])`

<small> added in 1.4.0 </small>

- `bot` - <[Object][object]>. [Telegraf](https://telegraf.js.org/classes/Telegraf.html) instance.
- `plugins` - <[Array][array]>. Array of [plugins](./plugin) which will be enabled.

## methods

### enable(launchImmediate)

<small> added in 1.4.0 </small>

Enable analytics.
If `launchImmediate` is `true` - also calls `bot.launch()` under the hood.

### enable(stopImmediate)

<small> added in 1.4.0 </small>

Disable analytics.
If `stopImmediate` is `true` - also calls `bot.stop()` under the hood.
