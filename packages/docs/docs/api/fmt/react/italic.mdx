# Italic API

Mark input text as _bold_

<small>added in 1.5.0</small>

## Import

```jsx
import Italic from "@tlgr/fmt/react/italic";
// or
import { Italic } from "@tlgr/fmt/react";
```

## Props

| Name     | Type | Default | Description                   |
| -------- | ---- | ------- | ----------------------------- |
| children | node |         | The content of the component. |
