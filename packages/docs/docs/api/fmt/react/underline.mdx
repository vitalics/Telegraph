# Underline API

returns <u>Underline</u> text

<small>added in 1.5.0</small>

## Import

```jsx
import Underline from "@tlgr/fmt/react/underline";
// or
import { Underline } from "@tlgr/fmt/react";
```

## Props

| Name     | Type | Default | Description    |
| -------- | ---- | ------- | -------------- |
| children | node |         | component text |
