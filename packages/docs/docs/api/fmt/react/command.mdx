# Command API

Add `/` to command name

<small>added in 1.5.0</small>

## Import

```jsx
import Command from "@tlgr/fmt/react/command";
// or
import { Command } from "@tlgr/fmt/react";
```

## Props

| Name | Type   | Default | Description   |
| ---- | ------ | ------- | ------------- |
| name | string |         | Command name. |
