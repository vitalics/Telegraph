# Bold API

Mark input text as **bold**

<small>added in 1.5.0</small>

## Import

```jsx
import Bold from "@tlgr/fmt/react/bold";
// or
import { Bold } from "@tlgr/fmt/react";
```

## Props

| Name     | Type | Default | Description                   |
| -------- | ---- | ------- | ----------------------------- |
| children | node |         | The content of the component. |
