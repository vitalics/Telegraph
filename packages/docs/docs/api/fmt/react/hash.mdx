# Hash API

Add `#` to name

<small>added in 1.5.0</small>

## Import

```jsx
import Hash from "@tlgr/fmt/react/hash";
// or
import { Hash } from "@tlgr/fmt/react";
```

## Props

| Name | Type   | Default | Description |
| ---- | ------ | ------- | ----------- |
| hash | string |         | Hash name.  |
