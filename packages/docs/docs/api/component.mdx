# Inline Component

Abstract class which implements Node.js [EventEmitter](https://nodejs.org/dist/latest-v16.x/docs/api/events.html#class-eventemitter) class,
but provides only `on`, `once`, `emit` functions.

```jsx title="example.ts"
import { InlineComponent } from "@tlgr/component";
class MyComponent extends InlineComponent {
  render() {
    // returns InlineKeyboardButton from telegram API
    // e.g. Markup.button.callback('my button');
  }
}
```

## `constructor(bot)`

<small> added in 1.0.0 </small>

- `bot` - <[Object][object]>. [Telegraf](https://telegraf.js.org/classes/Telegraf.html) instance.

## methods

### `on(event, callback)`

<small> added in 1.0.0 </small>

- `event` - <[string][string]>. Event name.
- `callback` - <[Function][function]>. Callback function.

### `once(event, callback)`

<small>added in 1.0.0</small>

- `event` - <[string][string]>. Event name.
- `callback` - <[Function][function]>. Callback function

### `emit(event, [...args])`

<small>added in 1.0.0</small>

- `event` - <[string][string]>. Event name.
- `args` - <[Array][array]>. Emit arguments, this args will comes to `callback` function for `on` and `once` event subscription.
