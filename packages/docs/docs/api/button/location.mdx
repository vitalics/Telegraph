# Location

Location button is a keyboard only button. After clicking - ask app to share location with bot.

After location recieved - fires [`location`](#location-1) event.

## extends

[KeyboardButton](./keyboard)

## Events

### Click

<small>added in 1.1.0</small>

#### name

`click`

#### payload

- `ctx` - <[Object][object]>. Telegraf update [Context](https://telegraf.js.org/classes/Context.html#addStickerToSet).
- `button` - <[Object][object]> - the [Keyboard Button](./keyboard) instance

### Location

<small>added in 1.1.0</small>

#### name

`click`

#### payload

- `ctx` - <[Object][object]>. Telegraf update [Context](https://telegraf.js.org/classes/Context.html#addStickerToSet).
- `location` - <[Object][object]> - the [location](https://telegraf.js.org/modules/Types.html#ExtraLocation) object
