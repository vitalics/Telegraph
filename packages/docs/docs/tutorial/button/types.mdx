# Button types

Button can request location, phone or contact. This components can be rendered only as `keyboard`, not `inline_keyboard`.

## Phone

Requests phone number.

## Poll

Ask telegram to create new poll on telegram UI.

If you would like to create Poll as a code - see `@tlgr/poll` package

## Quiz

Ask telegram to create new poll on telegram UI.

If you would like to create Quiz as a code - see `@tlgr/quiz` package
