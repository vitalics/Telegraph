The new telegraph components are landed and available on [`npm.js`](https://npmjs.com) site.

## New Components:

### [Carousel](https://github.com/vitalics/telegraph/packages/carousel/)

render item and navigation (`<` and `>`) symbols.

Can render `text` , `video`, `audio`, `animation` and `photo` messages.

Installation:

`npm i @tlgr/carousel`

### [Poll](https://github.com/vitalics/telegraph/packages/poll/)

Poll ask user to answer a question.

Installation:

`npm i @tlgr/poll`

### [Quiz](https://github.com/vitalics/telegraph/packages/poll/)

Quiz is similar to poll but have `correct_option_id` which marks correct answer id.

Installation:

`npm i @tlgr/quiz`

## Changes

### [Button](https://github.com/vitalics/telegraph/packages/button)

Introduce button keyboard component.

New Keyboard buttons. See [types](https://vitalics.github.io/Telegraph/docs/tutorial/button/types):

- location
- phone
- poll
- quiz

### [Component](https://github.com/vitalics/telegraph/packages/component)

New Component: Keyboard
