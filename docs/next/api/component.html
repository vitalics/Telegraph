Inline Component

Abstract class which implements Node.js EventEmitter class, but provides only on, once, emit functions.

example.ts
import { InlineComponent } from "@tlgr/component";
class MyComponent extends InlineComponent {
  render() {
    // returns InlineKeyboardButton from telegram API
    // e.g. Markup.button.callback('my button');
  }
}

constructor(bot)​

added in 1.0.0
  • bot - <Object>. Telegraf instance.

methods​

on(event, callback)​

added in 1.0.0
  • event - <string>. Event name.
  • callback - <Function>. Callback function.

once(event, callback)​

added in 1.0.0
  • event - <string>. Event name.
  • callback - <Function>. Callback function

emit(event, [...args])​

added in 1.0.0
  • event - <string>. Event name.
  • args - <Array>. Emit arguments, this args will comes to callback function for on and once event subscription.
